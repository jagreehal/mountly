## Run locally

From the repo root after `pnpm install && pnpm -r build`:

```bash
pnpm --filter mountly-docs run stage-examples
pnpm --filter mountly-docs dev
```

Per-example dev servers are documented in [`docs/examples/README.md`](https://github.com/jagreehal/mountly/blob/main/docs/examples/README.md).
